```
C) Virtual Protected Cloud
D) Virtual Premium Cloud
```

```
A) 256개

D) 16,777,216개
```

```
A) 인터넷 게이트웨이에 직접 연결된다

C) 외부에서 직접 접근할 수 없다
D) 라우팅 테이블이 필요하지 않다

```


```
A) 인바운드 트래픽 로드 밸런싱

```

## 문제 6

```
A) 보안 그룹은 Stateless, NACL은 Stateful
B) 보안 그룹은 Stateful, NACL은 Stateless
C) 둘 다 Stateful
D) 둘 다 Stateless
```

## 문제 7

```
A) Allow 규칙만 가능
```

```
A) 모든 규칙을 동시에 평가

D) 랜덤하게 규칙 선택
```

## 문제 9

```
A) 같은 리전 내에서만 가능
```

## 문제 10

```
A) 비용이 무료
```

## 문제 11
Site-to-Site VPN의 특징은?
```
A) 전용 물리적 연결
```

## 문제 12
