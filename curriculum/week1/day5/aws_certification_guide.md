# AWS Solutions Architect Associate 자격증 소개 및 시험 등록 방법

## 개요

AWS Solutions Architect Associate(SAA-C03) 자격증은 AWS 클라우드 인프라 설계 및 배포에 관한 지식과 기술을 검증하는 자격증입니다. 이 자격증은 클라우드 아키텍처 설계 분야에서 가장 인기 있는 자격증 중 하나로, AWS 클라우드 서비스를 활용하여 안전하고 견고한 애플리케이션을 설계하고 배포하는 능력을 증명합니다.

## 자격증 정보

### 시험 코드
- SAA-C03

### 시험 형식
- 선다형(정답 1개, 오답 3개) 및 복수 응답형(5개 이상 선택지 중 2개 이상 정답)
- 총 65문항 (채점 대상 50문항 + 채점 비대상 15문항)
- 시험 시간: 130분
- 합격 기준: 1000점 만점 중 720점 이상 (72%)
- 시험 비용: $150 USD (부가세 별도)
- 시험 언어: 영어, 일본어, 한국어, 중국어 등 다양한 언어 지원

### 도메인별 출제 비율
1. 보안 아키텍처 설계: 30%
2. 복원력을 갖춘 아키텍처 설계: 26%
3. 고성능 아키텍처 설계: 24%
4. 비용에 최적화된 아키텍처 설계: 20%

### 자격증 유효 기간
- 3년 (이후 갱신 필요)

## 시험 준비 방법

### 공식 학습 자료
1. [AWS 공식 시험 가이드](https://d1.awsstatic.com/training-and-certification/docs-sa-assoc/AWS-Certified-Solutions-Architect-Associate_Exam-Guide.pdf)
2. [AWS 샘플 문항](https://d1.awsstatic.com/training-and-certification/docs-sa-assoc/AWS-Certified-Solutions-Architect-Associate_Sample-Questions.pdf)
3. [AWS 백서 및 가이드](https://aws.amazon.com/whitepapers/)
4. [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/)

### 추천 학습 경로
1. AWS 공식 시험 가이드 검토
2. 본 커리큘럼의 모든 주차 학습 내용 완료
3. AWS 공식 샘플 문항 풀이
4. 모의고사 응시 (5주차에 제공)
5. AWS 백서 중 핵심 내용 검토

## 시험 등록 방법

### 1. AWS Certification 계정 생성

1. [AWS Certification 웹사이트](https://www.aws.training/certification)에 접속합니다.
2. "AWS 계정으로 로그인" 또는 "새 계정 생성" 버튼을 클릭합니다.
3. AWS 계정이 있는 경우 해당 계정으로 로그인하고, 없는 경우 새 계정을 생성합니다.
4. 개인 정보 및 연락처 정보를 입력합니다.
5. 계정 생성 후 이메일 인증을 완료합니다.

### 2. 시험 등록

1. AWS Certification 계정에 로그인합니다.
2. 대시보드에서 "시험 예약" 버튼을 클릭합니다.
3. "AWS Certified Solutions Architect - Associate"를 선택합니다.
4. 시험 언어를 선택합니다 (한국어 또는 영어 권장).
5. 시험 방식을 선택합니다:
   - **온라인 감독 시험**: 집이나 사무실에서 온라인으로 응시
   - **테스트 센터 시험**: 지정된 테스트 센터에서 응시

### 3. 시험 일정 및 장소 선택

1. 원하는 날짜와 시간을 선택합니다.
2. 테스트 센터 시험을 선택한 경우, 가까운 테스트 센터를 선택합니다.
3. 온라인 감독 시험을 선택한 경우, 시스템 요구 사항을 확인합니다.

### 4. 결제 및 확인

1. 시험 비용을 결제합니다 ($150 USD + 부가세).
2. 결제 확인 및 영수증을 이메일로 받습니다.
3. 시험 등록 확인 이메일을 저장합니다.

## 온라인 감독 시험 준비 사항

### 시스템 요구 사항
- 안정적인 인터넷 연결 (최소 1Mbps 이상)
- 웹캠이 있는 컴퓨터
- 마이크 및 스피커
- 지원되는 운영 체제 및 브라우저
- 관리자 권한이 있는 계정

### 시험 환경 요구 사항
- 조용하고 개인적인 공간
- 책상 위에 불필요한 물건 제거
- 벽에 메모나 포스터 등이 없어야 함
- 시험 중 다른 사람의 출입 금지
- 휴대폰 및 기타 전자 기기 접근 금지

### 신분증 요구 사항
- 정부에서 발급한 사진이 있는 신분증 (여권, 운전면허증 등)
- 이름이 AWS Certification 계정과 일치해야 함

## 테스트 센터 시험 준비 사항

### 지참물
- 정부에서 발급한 사진이 있는 신분증 (여권, 운전면허증 등)
- 시험 등록 확인 이메일 또는 등록 ID

### 주의 사항
- 시험 시작 15분 전에 도착
- 개인 소지품은 보관함에 보관
- 시험장 내 음식물 반입 금지
- 시험 규정 준수

## 시험 당일 체크리스트

### 온라인 감독 시험
- 시험 시작 30분 전에 로그인
- 시스템 점검 및 감독관과의 연결 설정
- 신분증 확인 및 작업 공간 검사
- 시험 규정 동의

### 테스트 센터 시험
- 시험 장소 및 시간 재확인
- 신분증 지참
- 시험장 규정 준수

## 시험 후 절차

### 결과 확인
- 시험 종료 직후 합격/불합격 여부 확인 가능
- 상세 점수 리포트는 이메일로 발송 (보통 5일 이내)

### 자격증 발급
- 합격 시 디지털 배지 및 자격증 발급
- AWS Certified 로고 사용 권한 부여
- AWS Certification 계정에서 자격증 확인 및 관리

### 불합격 시
- 14일 후 재응시 가능
- 3회 불합격 시 14일 대기 기간 후 재응시 가능
- 시험 점수 리포트를 통해 부족한 영역 파악 및 보완

## 자격증 취득의 이점

1. **취업 경쟁력 강화**: AWS 클라우드 전문가에 대한 수요 증가
2. **연봉 협상력 향상**: 자격증 소지자의 평균 연봉이 더 높음
3. **전문성 입증**: 클라우드 아키텍처 설계 능력 공식 인증
4. **AWS 커뮤니티 접근**: AWS 인증 전문가 커뮤니티 참여 기회
5. **지속적인 학습**: 클라우드 기술 트렌드에 대한 지속적인 학습 동기 부여

## 자주 묻는 질문 (FAQ)

### Q: 시험 준비에 얼마나 많은 시간이 필요한가요?
A: 개인의 경험과 배경지식에 따라 다르지만, 일반적으로 최소 80-120시간의 학습이 권장됩니다.

### Q: 실무 경험 없이 자격증을 취득할 수 있나요?
A: 가능하지만, AWS 서비스에 대한 실무 경험이 있으면 시험 준비와 합격에 도움이 됩니다.

### Q: 시험 언어를 영어가 아닌 다른 언어로 선택해도 되나요?
A: 네, 한국어를 포함한 여러 언어로 시험을 볼 수 있습니다. 단, 기술 용어는 영어로 표기되는 경우가 많습니다.

### Q: 시험 일정을 변경하거나 취소할 수 있나요?
A: 시험 24시간 전까지 일정 변경 또는 취소가 가능합니다. 그 이후에는 환불이 불가능합니다.

### Q: 자격증 갱신은 어떻게 하나요?
A: 자격증 만료 3년 이내에 재시험을 통해 갱신하거나, 상위 자격증을 취득하면 자동으로 갱신됩니다.

## 결론

AWS Solutions Architect Associate 자격증은 클라우드 컴퓨팅 분야에서 가치 있는 자격증으로, 취업 및 경력 발전에 큰 도움이 됩니다. 본 커리큘럼을 통해 체계적으로 학습하고, 실습을 통해 실무 경험을 쌓으면 자격증 취득에 성공할 수 있습니다. 시험 등록을 완료하고 목표 날짜를 정하는 것이 학습 동기 부여에 도움이 되므로, 가능한 빨리 시험 일정을 예약하는 것을 권장합니다.
